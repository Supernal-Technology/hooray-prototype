# Hooray Agency · SAL prototype

Client-facing demo of SAL — a hotel client opens their monthly report and talks to
their own data. Editorial report + "Talk to this data" chat (with inline charts),
plus a secondary AM view (portfolio, approvals, Ask SAL).

**Directional prototype · all data simulated · no backend.**

Live: https://supernal-technology.github.io/hooray-prototype/

## Local
```
npm install
npm run dev
```
Source of truth lives in the `client-demos` monorepo (`sal-dashboard/`); this repo
is the public, Pages-deployed mirror.
